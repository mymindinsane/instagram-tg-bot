# Instagram TG Bot (Java, Maven)

Телеграм-бот на Java (Maven) для сравнения списков Instagram: взаимные, кто не подписан в ответ, и кого не фоллоуешь ты. Без БД.

## Как это работает
- Бот не использует официальные API Instagram.
- На старте режим без парсинга: ты загружаешь два списка никнеймов (followers и following) в чат бота — каждым по одному нику в строке, как .txt/.csv или текстом.
- Бот вычисляет взаимные и расхождения, возвращает файлы `mutuals.txt`, `not_following_back.txt`, `not_followed_by_you.txt`.

## Запуск
1. Установи переменную окружения:
   - `TELEGRAM_BOT_TOKEN` — токен твоего бота
2. Собери и запусти:
```bash
mvn -q -e -DskipTests package
java -jar target/instagram-tg-bot-0.1.0-shaded.jar
```

## Использование
- Команда `/start` — инструкция.
- Команда `/check` — начать новую проверку, затем отправь два файла: сначала followers, потом following.
- Можно вместо файла просто прислать текстом по одному нику в строке.
